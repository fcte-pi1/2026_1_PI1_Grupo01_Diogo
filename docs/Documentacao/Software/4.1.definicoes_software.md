## Definições

### Front-end e Interface Web

> Esta tarefa consiste em definir a tecnologia da interface web responsável por exibir informações de telemetria do Micromouse, como trajeto percorrido, nível de bateria e velocidade em tempo real. (com justificativa)

### Tecnologias do Back-end e Algoritmo do MicroMouse

> Esta tarefa consiste nas definições iniciais relacionadas à arquitetura do back-end do Micromouse. Inclui a elaboração do modelo de dados, a escolha das tecnologias que compõem o servidor e os serviços de apoio, além da seleção do simulador que será utilizado para testes do código. Também contempla a definição do algoritmo responsável pela lógica de navegação do robô.

### Banco de Dados

> Consiste em selecionar o banco de dados destinado ao armazenamento histórico das corridas e às consultas por labirinto.
